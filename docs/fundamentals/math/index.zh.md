# 机器学习数学基础

在掌握机器学习之前，理解支撑这些算法的基础数学概念至关重要。

## 主要内容

### 线性代数

线性代数对于理解许多算法尤其是深度学习中使用的算法至关重要。主要概念包括：

* 向量
* 矩阵
* 行列式
* 特征值和特征向量
* 向量空间
* 线性变换

### 微积分

许多机器学习算法涉及连续函数的优化，这需要理解：

* 导数
* 积分
* 极限
* 多元微积分
* 梯度概念

### 概率与统计

这些对于理解模型如何从数据中学习和做出预测至关重要。关键概念包括：

* 概率论
* 随机变量
* 概率分布
* 期望
* 方差
* 协方差
* ...
